Installation Guide and Applications Used:

1. Install XAMPP

2. Text Editor - Visual Studio Code.

3. Download the zip file

4. Extract the file in root directory where you installed xammp local disk C: or drive D: or drive E: paste: (for xampp/htdocs)

5. Open PHPMyAdmin (http://localhost/phpmyadmin)

6. Create a database with name lougeh

6. Import lougeh.sql file(given inside the zip package in SQL file folder)

7.Run the script (prefer in google chrome) http://localhost/BFIassessment/index.php


Note: port number in drall.php, orall.php, orprint.php must be changed to the port where the xampp/web server is installed to be able to generate pdf.
